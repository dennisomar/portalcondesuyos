# Portal Condesuyos

Este es el portal web oficial sobre la **provincia de Condesuyos - Arequipa**, diseñado para ser una referencia histórica, cultural y turística.

## 🚀 Características principales
- Historia, cultura, turismo y recursos de Condesuyos.
- Páginas dedicadas a cada distrito.
- Contador global de visitas.
- Formulario de comentarios (almacenado local).
- Galería de fotos inicial.
- Calendario de fiestas y eventos.
- Mapa interactivo con LeafletJS.
- Sección "Orgullo Condesuyos".

## 🌍 Despliegue en GitHub Pages
1. Crea un repositorio en GitHub (ej. `portal-condesuyos`).
2. Sube todos los archivos con:
   ```bash
   git init
   git remote add origin https://github.com/usuario/portal-condesuyos.git
   git add .
   git commit -m "Portal Condesuyos inicial"
   git push -u origin main
   ```
3. Entra a `Settings > Pages` y activa GitHub Pages desde la rama `main`.
4. Tu portal estará en: `https://usuario.github.io/portal-condesuyos/`

## 🚀 Alternativa: Despliegue en Netlify o Vercel
- Entra a [Netlify](https://netlify.com) o [Vercel](https://vercel.com).
- Conecta tu repo de GitHub.
- En segundos tendrás tu portal publicado gratis.
